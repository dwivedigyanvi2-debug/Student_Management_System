# 🎓 Student Management System

A simple **Student Management System** built using **C++** as a beginner project to practice programming concepts like functions, conditions, loops, file handling (optional), and object-oriented programming.

## 📌 Project Description

This project helps manage student records by allowing users to perform basic operations such as:

* Add student details
* View student records
* Search student information
* Update student data
* Delete student records

This project was created as a **1st-year Computer Science student project** to improve problem-solving and programming skills.

---

## 🚀 Features

✔️ Add new student details
✔️ Display all student records
✔️ Search student by ID or Name
✔️ Update existing student information
✔️ Delete student records
✔️ Simple menu-driven interface
